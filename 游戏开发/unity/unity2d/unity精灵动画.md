## unity精灵动画

##### 1.Animation

###### 1. 制作界面

> 选中待提添加动画的物体，然后点击 Window ->  Animation -> Animation
>
> 如果选中的物体无Animation/Animator组件，会自动添加Animator组件

###### 2.制作完成的Animation

![image-20241112205154342](../../assets/image-20241112205154342.png)

| 名称         | 介绍                                 |
| ------------ | ------------------------------------ |
| Length       | 长度（动画时长）                     |
| Loop Time    | 循环                                 |
| Loop Pose    | 循环动作（使循环时头部尾部衔接平滑） |
| Cycle Offset | 平滑度                               |
